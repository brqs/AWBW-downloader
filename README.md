# AWBW_downloader

#### 介绍
AWBW对局下载器。

#### 参考运行环境
1.  python 3.6.4
2.  bs4 4.6.0
3.  cv2 4.1.1
4.  numpy 1.18.5
5.  requests 2.23.0

#### 使用说明
将主程序中new_list中的内容修改为想要下载的对局id然后运行即可。     
对局文件将自动生成于程序运行目录内，也可以设置复制到某个位置，修改主程序中dest_dir变量即可。   
由于函数文档尚未编写，其余功能请联系笔者。
